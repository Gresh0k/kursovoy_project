# АИС «АвтоМойка» — постамат v1.1

Программное обеспечение **сенсорного постамата** автомойки самообслуживания  
ИП «АвтоМойка Самообслуживания», г. Ижевск.

> **Версия:** 1.1 — клиентский киоск + панель оператора  
> **Ветка Git:** `release/v1.1`  
> **Версия 1.0** (только клиентский киоск) — в ветке [`release/v1.0`](../tree/release/v1.0)

---

## Описание

Расширение v1.0: добавлены **подтверждение входа по PIN-коду**, **админ-панель** для персонала и отдельная база данных. Клиентская часть (мойка, бонусы, пополнение) сохранена и доработана.

**Тариф:** 1 ₽ = 1 секунда общего таймера.

## Возможности

### Гость (как в v1.0)
- Пополнение (пресеты или циферблат), оплата картой (имитация)
- Сессия мойки с выбором услуг

### Зарегистрированный клиент
- Вход по телефону → **PIN-код** (4 цифры)
- Регистрация с установкой PIN (ввод + повтор)
- Оплата бонусами до **100%**, списание с баланса
- Проверка блокировки аккаунта

### PIN вместо SMS (бесплатно)
- При регистрации клиент задаёт **4-значный код**
- При каждом входе — телефон → код на циферблате
- Быстро, без интернета и без платы за SMS  
- *В промышленной версии PIN можно заменить на SMS-API (SMS.ru и т.п.)*

### Сессия мойки
- Таймер стартует после выбора первой услуги
- Пауза до 60 сек, переключение услуг
- Досрочное завершение → бонусы на счёт клиента
- Крупный таймер **«Осталось»**, мигание за **30 сек** до конца

### Панель оператора (админ)
**Вход:** на экране «Вход по телефону» ввести **секретный номер** и **служебный PIN** (настраиваются в `config.ini`). Для клиента это выглядит как обычный запрос кода.

**Разделы:**
- **Бесплатная мойка** — запуск сессии без оплаты (60 / 120 / 300 / 600 сек)
- **Клиенты** — список, блокировка / разблокировка, **установка бонусов** с циферблата

### Интерфейс
- Полноэкранный сенсорный режим (tkinter)
- Встроенные диалоги (без системных окон Windows)
- Таймаут бездействия **60 сек** на экранах ввода → главная  
  *(не действует во время мойки, на экране завершения и в админке)*

## Установка и запуск

### 1. Клонировать репозиторий (ветка v1.1)

```bash
git clone <URL-репозитория>
cd <имя-репозитория>
git checkout release/v1.1
```

### 2. Запуск

Из **корня репозитория**:

```bash
python -m avtomoyka_v2.main
```

Или двойной клик по `run_v2.bat` в корне проекта.

База данных создаётся автоматически: `avtomoyka_v2/avto_v2.db`.

### 3. Отладка

- `Esc` — выход из полноэкранного режима
- В `config.ini`: `fullscreen = false` — окно вместо полного экрана

---

## Настройки (`config.ini`)

```ini
[database]
path = avto_v2.db

[app]
version = 1.1

[kiosk]
fullscreen = true
idle_timeout_seconds = 60

[tariff]
rub_per_second = 1
pause_budget_seconds = 60
session_warning_seconds = 30

[auth]
admin_phone = 88005553535
admin_pin = 2026
pin_length = 4
```

| Параметр | Описание |
|----------|----------|
| `admin_phone` | Секретный номер для входа операторов |
| `admin_pin` | Служебный 4-значный код |
| `pin_length` | Длина PIN клиента (по умолчанию 4) |
| `idle_timeout_seconds` | Бездействие → главная (экраны ввода) |
| `session_warning_seconds` | Мигание таймера за N сек до конца |

> **Важно:** смените `admin_phone` и `admin_pin` перед эксплуатацией. Значения по умолчанию — только для демонстрации.
