# АИС «АвтоМойка» — постамат v1.0

---

## Описание

Приложение устанавливается на сенсорный терминал у поста мойки. Клиент пополняет баланс, выбирает услуги (вода, пена, осмос, воск, пылесос) и моет автомобиль в рамках купленного времени.

**Тариф:** 1 ₽ = 1 секунда общего таймера.

---

## Возможности v1.0

### Гость
- Пополнение суммы (пресеты 100 / 200 / 300 / 500 ₽ или ввод с циферблата)
- Оплата картой (имитация)
- Запуск сессии мойки

### Зарегистрированный клиент
- Вход по номеру телефона
- Регистрация при первом входе
- Оплата бонусами до **100%** (бонусы — остаток с прошлой досрочной мойки)
- Списание с предоплаченного баланса
- Блокировка аккаунта (проверка при входе)

### Сессия мойки
- Таймер **не идёт**, пока не выбрана первая услуга
- Переключение услуг в любой момент
- **Пауза** — до 60 сек за сессию (выход из паузы — нажатие на услугу)
- **Досрочное завершение** — неиспользованные секунды → на бонусный счёт клиента
- Крупный таймер «Осталось», **мигание** за 30 сек до конца

### Интерфейс
- Полноэкранный сенсорный режим (tkinter)
- Таймаут бездействия **60 сек** на экранах ввода → возврат на главную

## Требования

| Компонент | Версия |
|-----------|--------|
| Python | 3.11+ |
| ОС | Windows 10/11 |
| Библиотеки | только стандартная (tkinter, sqlite3) |

При установке Python включите компонент **tcl/tk and IDLE** (для tkinter).

## Установка и запуск

### 1. Клонировать репозиторий (ветка v1.0)

```bash
git clone <URL-репозитория>
cd <имя-репозитория>
git checkout release/v1.0
```

### 2. Запуск

Из **корня репозитория**:

```bash
python -m avtomoyka.main
```

Или двойной клик по `avtomoyka/run.bat`.

База данных создаётся автоматически: `avtomoyka/avto.db`.

### 3. Отладка

- `Esc` — выход из полноэкранного режима
- В `avtomoyka/config.ini` можно отключить полный экран:

```ini
[kiosk]
fullscreen = false
```

---

## Настройки (`config.ini`)

```ini
[database]
path = avto.db

[app]
version = 1.0

[kiosk]
fullscreen = true
idle_timeout_seconds = 60

[tariff]
rub_per_second = 1
pause_budget_seconds = 60
session_warning_seconds = 30
```

| Параметр | Описание |
|----------|----------|
| `rub_per_second` | 1 ₽ = N сек мойки |
| `pause_budget_seconds` | Общий запас паузы за сессию |
| `session_warning_seconds` | За сколько секунд до конца таймера мигать |
| `idle_timeout_seconds` | Бездействие на экранах ввода → главная |

## Структура проекта

```
avtomoyka/
├── main.py                 # Точка входа
├── config.ini              # Настройки
├── run.bat                 # Запуск под Windows
├── data/
│   └── db.py               # SQLite: клиенты, услуги, сессии
├── services/
│   ├── client_service.py   # Клиенты, баланс, бонусы
│   └── session_service.py  # Сессии, таймер
└── ui/
    ├── kiosk_app.py        # Навигация между экранами
    ├── dialogs.py          # Встроенные диалоги
    ├── idle_timer.py       # Таймаут бездействия
    ├── widgets.py          # Кнопки, циферблат
    └── screens/
        ├── welcome.py      # Главный экран
        ├── topup.py        # Пополнение / оплата
        ├── phone.py        # Вход по телефону
        ├── session.py      # Мойка
        └── finish.py       # Завершение
```
